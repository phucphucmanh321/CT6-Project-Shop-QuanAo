## Yêu Cầu Ảnh

- **Định dạng**: JPG, PNG, hoặc WebP
- **Kích thước**: 400x500px hoặc lớn hơn (sẽ được scale tự động)
- **Chất lượng**: Cao (ít nhất 1 MB mỗi file)

## Cách Thêm Ảnh

1. Copy/paste ảnh vào thư mục này
2. Đặt tên theo tên file ở trên
3. App sẽ tự động load ảnh từ thư mục này

## Ví Dụ Cấu Trúc

```
src/assets/images/products/
├── product-1-big-ss-long-tee-black.jpg
├── product-2-striped-sweatshirt-navy.jpg
├── product-3-striped-long-sleeve-tee.jpg
├── ...
└── product-20-polo-shirt-black.jpg
```

Sau khi thêm ảnh xong, refresh trình duyệt để xem kết quả!
